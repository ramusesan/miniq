Getting Started

Installation

Include the MiniQ library in your HTML file:

<script src="miniq.js"></script>

Basic Usage

Use the MQ() function to select elements and perform operations:

// Select elements by CSS selector
MQ('.my-class').addClass('active');

// Chain methods
MQ('#my-element')
  .css('color', 'blue')
  .html('New content')
  .on('click', function() {
    alert('Element clicked!');
  });

Core Concepts

Namespace Structure

MiniQ is organized with the following structure:

  • MQ() or MiniQ() - Main selector function (alias to MiniQ.init)
  • MQ.fn - Access to the full MiniQ namespace
  • MQ.fn.Collection - The constructor for element collections
  • MQ.utils - Utility functions

Collection Pattern

MiniQ uses a collection pattern to manage DOM elements. All element methods operate on every element in the collection and return the collection for chaining.

Note: Most methods in MiniQ return the collection object, allowing you to chain multiple methods together for concise code.

API Reference

Selectors

Select elements using CSS selectors:

// Select by CSS selector
MQ('.my-class');
MQ('#my-id');
MQ('div.container');

// Create a collection from a DOM element
MQ(document.getElementById('myElement'));

// Create a collection from an array of elements
MQ([elem1, elem2, elem3]);

DOM Manipulation

Classes

Method Description
addClass(className) Adds a class to all elements in the collection
removeClass(className) Removes a class from all elements in the collection
toggleClass(className) Toggles a class on all elements in the collection
hasClass(className) Returns true if any element in the collection has the class 
// Add, remove, toggle, or check for classes
MQ('.box').addClass('active');
MQ('.box').removeClass('active');
MQ('.box').toggleClass('active');
const hasClass = MQ('.box').hasClass('active'); // returns boolean

Styles

Method Description
css(property) Gets the computed style value for the first element
css(property, value) Sets a style property on all elements
css(propertiesObject) Sets multiple style properties on all elements 
// Get computed style
const color = MQ('#element').css('color');

// Set a single style
MQ('#element').css('color', 'blue');

// Set multiple styles
MQ('#element').css({
  'color': 'blue',
  'background-color': '#eee',
  'padding': '10px'
});

Content

Method Description
html() Gets the HTML content of the first element
html(content) Sets the HTML content of all elements
text() Gets the text content of the first element
text(content) Sets the text content of all elements 
// Get HTML content
const html = MQ('#element').html();

// Set HTML content
MQ('#element').html('<strong>New content</strong>');

// Get text content
const text = MQ('#element').text();

// Set text content
MQ('#element').text('New text');

Attributes

Method Description
attr(name) Gets an attribute value from the first element
attr(name, value) Sets an attribute on all elements
attr(attributesObject) Sets multiple attributes on all elements
removeAttr(name) Removes an attribute from all elements 
// Get attribute
const href = MQ('a').attr('href');

// Set attribute
MQ('a').attr('href', 'https://example.com');

// Set multiple attributes
MQ('a').attr({
  'href': 'https://example.com',
  'target': '_blank',
  'rel': 'noopener'
});

// Remove attribute
MQ('a').removeAttr('target');

Visibility

Method Description
show() Shows all elements (sets display to 'block')
hide() Hides all elements (sets display to 'none')
toggle() Toggles visibility of all elements 
// Show elements
MQ('.box').show();

// Hide elements
MQ('.box').hide();

// Toggle visibility
MQ('.box').toggle();

Collection Utilities

Method Description
each(callback) Executes a function for each element in the collection
size() Returns the number of elements in the collection
get(index) Returns the element at the specified index
toArray() Converts the collection to a regular array 
// Get collection size
const count = MQ('div').size();

// Get a specific element
const secondDiv = MQ('div').get(1);

// Convert to array
const divArray = MQ('div').toArray();

// Iterate over elements
MQ('div').each(function(el, index) {
  console.log(index, el);
});

Event Handling

Method Description
on(event, callback) Adds an event listener to all elements
off(event, [callback]) Removes event listener(s) from all elements
click(callback) Adds a click event listener to all elements
click() Triggers a click event on all elements 
// Add event listener
MQ('#button').on('click', function(e) {
  console.log('Button clicked!');
});

// Remove event listener
MQ('#button').off('click', handlerFunction);

// Remove all listeners for an event
MQ('#button').off('click');

// Click shorthand
MQ('#button').click(function() {
  console.log('Button clicked!');
});

// Trigger click event
MQ('#button').click();

DOM Traversal

Method Description
find(selector) Finds elements matching the selector within the current elements
parent() Gets the parent elements of the current elements
children() Gets the child elements of the current elements 
// Find descendants
MQ('.container').find('p');

// Get parent elements
MQ('p').parent();

// Get child elements
MQ('.container').children();

AJAX

Method Description
MQ.ajax(options) Performs an AJAX request with the specified options
MQ.get(url, success) Performs a GET request
MQ.post(url, data, success) Performs a POST request 
// GET request
MQ.get('https://api.example.com/data', function(response) {
  console.log(response);
});

// POST request
MQ.post('https://api.example.com/data', 
  { name: 'John', age: 30 },
  function(response) {
    console.log(response);
  }
);

// Advanced AJAX options
MQ.ajax({
  method: 'POST',
  url: 'https://api.example.com/data',
  data: JSON.stringify({ name: 'John', age: 30 }),
  headers: {
    'Content-Type': 'application/json'
  },
  success: function(response) {
    console.log('Success:', response);
  },
  error: function(xhr, status) {
    console.error('Error:', status);
  },
  complete: function() {
    console.log('Request completed');
  }
});

Extending MiniQ

You can extend MiniQ with custom methods:

// Add a new method to all MiniQ collections
MQ.fn.Collection.prototype.highlight = function(color) {
  return this.each(function(el) {
    const originalColor = MQ(el).css('background-color');
    
    // Store original color
    el._originalBgColor = originalColor;
    
    // Set highlight color
    MQ(el).css('background-color', color || '#ffff99');
  });
};

// Add a method to unhighlight
MQ.fn.Collection.prototype.unhighlight = function() {
  return this.each(function(el) {
    if (el._originalBgColor) {
      MQ(el).css('background-color', el._originalBgColor);
    }
  });
};

// Usage
MQ('p').highlight('#ffcc00');
setTimeout(() => MQ('p').unhighlight(), 2000);

Examples

Basic DOM Manipulation

// Select all paragraphs and add a class
MQ('p').addClass('enhanced-text');

// Change styles on all list items
MQ('li').css({
  'color': '#333',
  'padding': '5px',
  'border-bottom': '1px solid #eee'
});

// Update content for a specific element
MQ('#status-message').html('<strong>Updated!</strong>');

Event Handling

// Add click handler to buttons
MQ('.action-btn').on('click', function() {
  // 'this' refers to the clicked element
  MQ(this).addClass('clicked');
  
  // Find related content
  const contentId = MQ(this).attr('data-content-id');
  MQ('#' + contentId).show();
});

// Form submission
MQ('#contact-form').on('submit', function(e) {
  e.preventDefault();
  
  // Get form data and send
  const name = MQ('#name-input').val();
  const email = MQ('#email-input').val();
  
  MQ.post('/submit-contact', { name, email }, function(response) {
    MQ('#form-result').html('Thank you for your submission!');
  });
});

Complete Example: Tabbed Interface

// Simple tabs implementation
function initTabs() {
  // Hide all tab content initially except the first
  MQ('.tab-content').hide();
  MQ('.tab-content').get(0) && MQ(MQ('.tab-content').get(0)).show();
  
  // Mark first tab as active
  MQ('.tab').get(0) && MQ(MQ('.tab').get(0)).addClass('active');
  
  // Handle tab clicks
  MQ('.tab').on('click', function() {
    // Remove active class from all tabs
    MQ('.tab').removeClass('active');
    
    // Add active class to clicked tab
    MQ(this).addClass('active');
    
    // Hide all tab content
    MQ('.tab-content').hide();
    
    // Show the associated tab content
    const tabId = MQ(this).attr('data-tab-id');
    MQ('#' + tabId).show();
  });
}

// Initialize tabs when DOM is ready
initTabs();
